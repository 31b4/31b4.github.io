## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/31b4/31b4.github.io.git
```

2. Open `index.html` in your browser or serve locally:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .
```
